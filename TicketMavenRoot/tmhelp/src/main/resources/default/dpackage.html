Discount Packages

A Discount Package is a group of shows that can be sold as a unit in a single ticket request. A price can be set for the package that is lower than the total of the regular ticket prices for the individual shows.

Discount Packages are entered via the Package Manager.

A Package can contain any number of shows, and a show can be included in more than one package.

The program will keep track of the discounted ticket prices offered in a package and will print these discounted prices on any tickets sold in a package. In the case of a sold out show, the unfulfilled ticket requests will show the discounted ticket price so that the proper refund amount can be determined.

When the lottery is run, the customer will be entered for each show individually and if the auditorium is sold out, they may very well not get all of the shows they requested in the package, however the package price still remains.
